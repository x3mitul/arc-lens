# Changelog

All notable changes to the **ARC Lens** extension will be documented in this file.

## [0.1.0] - 2026-06-13

### Added
- Renamed project to ARC Lens.
- Integrated high-fidelity visual monogram logo.
- Replaced pan buttons with mouse-drag panning via `chartjs-plugin-zoom`.
- Implemented "Reset Zoom" feature that pauses auto-scroll during manual inspection.
- Added automatic Python backend dependency (`arc-training`) check and pip auto-installer prompt.
- Added comprehensive publishing guide and packaged local `.vsix` file.
